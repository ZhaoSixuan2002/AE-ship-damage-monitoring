## Usage

These animations are referenced in the paper using footnotes. They demonstrate:
1. The gradual evolution of the damage suspicion index over time
2. The spatial distribution of suspicion values on the 3D ship structure
3. The contrast between damaged and healthy scenarios
4. The effectiveness of temporal smoothing in suppressing false alarms
